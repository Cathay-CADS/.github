# Cathay-CADS

### Repository Name
Platform_ProjectName_{Description}_Framework_{Types}

| | |
|--|--|
|Platform | 使用平台 |
|ProjectName | 專案名稱（簡要說明Repo內容）|
|Description | 預留欄位，如REPO名稱相同時區分使用|
|Framework | 執行框架 |
|Type | 專案類別: POC、Tool，default為內部開發專案

e.g. GCP_ApiTemplate_API, GCP_CloudSQL_API_POC
